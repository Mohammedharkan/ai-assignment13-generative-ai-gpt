# AI Assignment 13: Introduction to Generative AI and GPT Text Generation

## Overview

This project demonstrates the fundamentals of Generative Artificial Intelligence using a pre-trained GPT model (DistilGPT2) from the Hugging Face Transformers library.

The notebook explores how GPT models generate text and how different generation parameters affect the output, including output length, temperature, Top-k, Top-p, and prompt engineering.

---

## Objectives

- Understand the basics of Generative AI.
- Explore the GPT architecture and text generation process.
- Load a pre-trained GPT model and tokenizer.
- Generate text using different prompts.
- Compare the effects of output length, temperature, Top-k, and Top-p.
- Demonstrate a simple content creation application.
- Discuss ethical considerations of Generative AI.

---

## Results

The experiments demonstrated that:

- Increasing **max_length** generates longer responses.
- Lower **temperature** produces more predictable text.
- Higher **temperature** generates more creative responses.
- **Top-k** and **Top-p** influence how the model selects words.
- Better prompts produce more relevant and higher-quality generated text.
